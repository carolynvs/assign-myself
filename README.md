# assign-myself

> A GitHub App built with [Probot](https://github.com/probot/probot) that Assign yourself to an issue with a comment

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
